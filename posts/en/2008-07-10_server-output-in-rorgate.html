I've added a developer menu to the resulting apps in RoRGate, if you download the 0.3 beta version you will see the change: With that, you as developer can use it to see your app in action, see the server output and edit it in TextMate. The only thing is that the server output works strange with 2.x rails apps, I don't know why, but only the error output is displayed in such rails apps. I use the NSTask class with an NSPipe for both the standart output and error but with rails 2.x it behaves strangely.
